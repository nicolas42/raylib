# Compile and Run

1. Install visual studio

2. Open cmd.exe 

3. Go to a folder that has a build.bat in it

4. Enter this command or wherever vcvarsall.bat is now  
"C:\Program Files\Microsoft Visual Studio\2022\Community\VC\Auxiliary\Build\vcvarsall.bat" x64  (only once)

5. Run this line in cmd.exe
build.bat && Debug\zomg.exe
